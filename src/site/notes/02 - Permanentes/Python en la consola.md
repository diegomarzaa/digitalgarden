- Ejecución directa de un script: python myscript.py

- Comandos comunes:
	- -h: all commands (help)
	- -python3 -V: (version)

- Cuando se llama con `-m module-name`, el módulo dado se encuentra en la ruta del módulo Python y se ejecuta como un script
